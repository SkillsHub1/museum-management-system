# Museum Management System

A full-stack web application developed for managing museum exhibit records.

## Features
* **Full CRUD Functionality:** Create, Read, Update, and Delete museum records.
* **Database Management:** Relational MySQL database for organized data storage.
* **Modern UI:** Custom CSS interface designed for administrative efficiency.
* **Security:** Implemented PDO prepared statements to prevent SQL injection.

## Tech Stack
* **Backend:** PHP (PDO)
* **Database:** MySQL
* **Frontend:** HTML5, CSS3
